# Mechanism: [[Mechanisms/chat_system|Chat System]]

## Description
The Chat system facilitates coordination between survivors within the same town instance.

## Town Forums
Town Forums are persistent message boards designed for long-term coordination. Unlike the transient nature of the global chat, Forum threads and replies are saved within the town instance, allowing players to plan complex construction projects or share scavenging tips that remain accessible across sessions.

## Communication UX
The chat system includes several advanced features to speed up coordination:
- **Item/Tile Chat Linking**: Players can link items from their inventory or specific tiles directly in the chat window.
- **Clickable Tile References**: Clicking on a linked tile reference automatically moves the player's map view to that specific location, facilitating rapid tactical response and exploration planning.

## System Announcements
- **Midnight Wave Data**: At midnight, the total number of monsters spawned in the current wave is announced.
- **Town-Fall Summaries**: Comprehensive reports on player deaths and current expedition outcomes are explicitly posted in the town chat at the midnight transition.
