# Mechanism: [[Mechanisms/shortcuts|Keyboard Shortcuts]]

To improve efficiency during exploration and base management, the following keyboard shortcuts are available:

## Navigation & Map
| Key | Action | Description |
| :--- | :--- | :--- |
| <kbd>Space</kbd> | **Center Map** | Immediately centers the hex map on your player character. |
| <kbd>E</kbd> | **Scavenge** | Toggles (Starts/Stops) scavenging on your current tile. |

## Menus & UI
| Key | Action | Description |
| :--- | :--- | :--- |
| <kbd>T</kbd> | **Chat** | Opens or closes the Town Chat panel. |
| <kbd>K</kbd> | **Skills** | Opens or closes the permanent Skill progression menu. |
| <kbd>Z</kbd> | **Inventory** | Opens or closes your personal inventory and recipes. |
| <kbd>O</kbd> | **Objectives** | Opens or closes the current Quests/Expedition Objectives. |
| <kbd>N</kbd> | **Release Notes** | Opens or closes the latest Patch Notes panel. |
| <kbd>H</kbd> | **Help** | Opens the official community wiki (this site). |

## Chat Interaction
When the chat panel is open via <kbd>T</kbd>:
- <kbd>Enter</kbd>: Focuses the input field to begin writing a message.
- <kbd>Escape</kbd>: Closes the chat panel immediately.

---
> **Tip**: Mastery of these shortcuts allows survivors to respond rapidly to **[[Mechanisms/midnight_cycle|Midnight]]** threats and coordinate effectively via the chat.
