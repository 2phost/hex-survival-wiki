# Skill: [[Skills/sweeping_strikes|Sweeping Strikes]]

Sweeping Strikes improves a survivor's ability to dispatch multiple threats at once. It is a vital specialization for clearings large hordes efficiently.

## 📝 Description
"Each attack has a chance to kill **1 extra monster**: **5% per level** (up to 25%)."

## 📈 Progression
This skill provides a chance for a "bonus kill" on every attack action.

| Level | Extra Kill Chance | Training Time (Real-Time) |
| :--- | :---: | :--- |
| **Lvl 1** | **5%** | **1 Hour** |
| **Lvl 2** | **10%** | **24 Hours** |
| **Lvl 3** | **15%** | **7 Days** |
| **Lvl 4** | **20%** | **14 Days** |
| **Lvl 5** | **25%** | **30 Days** |

## 🔓 Prerequisites
- **[[Skills/weapon_maintenance|Weapon Maintenance]]**: Level 2
