# Skill: [[Skills/generator_operations|Generator Operations]]
- **Description**: Safely run and refuel the town generator.
- **Max Level**: 3
- **Effect**: Unlocks the **"Refuel"** option in the Base menu.
 - **Lvl 1**: [[Items/gasoline_canister|Gasoline]] usage (+25%).
 - **Lvl 2**: [[Items/biofuel_cell|Biofuel]] usage (+40%).
 - **Lvl 3**: [[Items/plasma_fuel_rod|Plasma]] usage (+60%).
- **Prerequisites**: [[Skills/field_engineering|Field Engineering]] Lvl 1.

## Unlocks
- *None (End of specialization tree)*
