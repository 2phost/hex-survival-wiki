# Item: [[Items/cleanse_vial|Cleanse Vial]]

A volatile purifier that can cleanse one corrupted tile when used on it.

## Where to Find
- *Cannot be found in the wilderness (Crafting/Production only).*

## Usage
### Consumption
Used directly on a corrupted tile to return it to its base biome state.

### Crafting
- Can be brewed using an [[Items/ancient_relic|Ancient Relic]], [[Items/chemical_sludge|Chemical Sludge]], and [[Items/water|Water]].

## Technical Information
- **Item ID**: `cleanse_vial`
- **Rarity**: Rare
- **Category**: Consumable
