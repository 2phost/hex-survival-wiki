# Solar Cell

![[assets/items/solar_cell.png|150]]


### Core Properties
- **Rarity**: rare
- **Category**: Junk
- **Description**: Crafted photovoltaic cell for harvesting sunlight. Used to build solar panels.

## Usage
- Deconstruct at Base for materials.
- Used to build [[base/constructions|Solar Panels]].

## Deconstruction Yields
> **Note**: Yields are probabilistic. Each material is rolled independently; at low **[[skills/salvage|Salvage]]** levels, failing to retrieve any materials is possible.

- **Primary Yields**: [[items/circuit_boards|Circuit Boards]] (30%), [[items/copper_wiring|Copper Wiring]] (50%), [[items/scrap_metal|Scrap Metal]] (30%)
- **Rare Yields**: [[items/research_material|Research Material]] (4%)
