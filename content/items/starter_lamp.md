# Starter Lamp
**Rarity**: common
**Category**: Light
**Description**: A basic starter lamp for new scavengers. Provides light for 2 days.

## Light Stats
- **Illumination Days**: 2
- **Radius**: 0 (current hex only)

## Note
- If the lamp expires during the [[mechanisms/time_and_power|Midnight]] transition while standing in an unlit hex, the player is immediately **"Consumed by Darkness."**
