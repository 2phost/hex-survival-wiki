# Mechanism: Base and Resources

## The [[mechanisms/world_and_tiles|Base]] Tile
The Base is the heart of the town. It provides inherent light and serves as the primary hub for resource storage and infrastructure development.

### "Enter Base" Menu
When a player is standing on the **[[mechanisms/world_and_tiles|Tile: Base]]** hex, a new **"Enter Base"** button appears on the main bottom menu. Clicking this opens a specialized management interface.

## 1. Town Status & Power
The top of the Base menu displays the town's current survival status.
- **Light & Survival Bar**: A percentage meter indicating the town's current power levels.
- **Survival Condition**: **At 0%, the town falls.** This is the primary game-over condition.
- **Night Use**: Displays the predicted power consumption for the midnight transition.

## 2. Town Resources (The Bank)
The Base contains a shared storage system known as the **Town Resources** section.
- **Shared Inventory**: Shows all resources currently stored in the base and their quantities.
- **Depositing**: Clicking items in the player's personal inventory while in the Base menu moves them to the **Bank**.
- **Withdrawing**: Players can pick items from the Bank to add them to their personal inventory.
